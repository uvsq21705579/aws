Régles du jeu

Le but du jeu est d’être le premier à poser tous ses dominos. Les autres joueurs qui non pas pu finir, marquent alors autant de points qu’indiqués sur leurs dominos restants. Si la partie est bloquée (aucun joueur ne peut ajouter de domino) chacun compte les points qui les restent en mains. Une partie peut se jouer sur plusieurs manches, le gagnant est celui qui totalise le moins de points.
